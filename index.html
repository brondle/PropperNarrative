A prototype of a text adventure based on Vladimir Propp's 31 story functions, written with Inform 7. You can read more about Vladimir Propp and the ideas behind the game here. A proper guide to playing the game is forthcoming, but you may find Andrew Plotkin and Lea Albaugh's short guide to interactive fiction commands helpful if you're new to IF.

This game is a work in progress. If you have any feedback or catch any bugs, feel free to reach out to me over email (brent L bailey at gmail) or social media.

A Propper Narrative was created with Inform and has IFID 9B648682-F384-4BE9-8EA8-489F86648B0C. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
